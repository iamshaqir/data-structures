In IntelliJ IDEA, you can quickly generate a **for loop** using **live templates**. Here's how:

1. **For `fori` (Basic For Loop):**
    - Type `fori` and press `Tab`.
    - IntelliJ will generate a basic **for loop** template:
      ```java
      for (int i = 0; i < ; i++) {
      }
      ```

2. **For `iter` (Enhanced For Loop):**
    - Type `iter` and press `Tab`.
    - IntelliJ will generate an **enhanced for loop** template:
      ```java
      for (Type item : collection) {
          
      }
      ```

3. **Customizing or Exploring More:**
    - Go to **File > Settings > Editor > Live Templates** to explore or create your own custom templates.

These shortcuts save time and help maintain consistent code structure.
